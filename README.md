# 🔐 Password Strength Checker

A Python-based password strength checker developed as part of my cybersecurity internship. This project evaluates the strength of a password based on common security requirements and provides useful suggestions for creating stronger passwords.

---

## 📌 Features

- ✅ Checks password length
- ✅ Detects uppercase letters
- ✅ Detects lowercase letters
- ✅ Detects numbers
- ✅ Detects special characters
- ✅ Calculates password score
- ✅ Classifies password as Weak, Medium, or Strong
- ✅ Provides suggestions to improve password strength
- ✅ Optimized implementation using a single loop
